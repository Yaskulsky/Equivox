## Build

```powershell
.\gradlew.bat build -x test
```

JAR: `build\libs\projecte-1.2.0.jar`

Opcjonalne integracje (JEI, Curios…) są wyłączone (`enable_optional_integrations=false` w `gradle.properties`).
